Descripción

Calculus Hub es una página web diseñada para facilitar la búsqueda y repaso de temas de Cálculo Diferencial e Integral.  
Incluye un buscador de conceptos, ejemplos claros y un flipbook integrado del libro *Cálculo de James Stewart* para consulta visual.

El sitio cuenta con un fondo animado estilo galaxia, lo que le da una apariencia moderna sin distraer del contenido principal.



Características principales

- Buscador rápido de temas y definiciones.  
- Ejemplos prácticos y secciones complementarias.  
- Flipbook interactivo con navegación tipo libro.  
- Fondo con partículas simuladas y colores suaves.  
- Diseño responsivo y adaptable a distintos dispositivos.
